Optimasi Algoritma Support Vector Machine Menggunakan Linear Discriminant Analysis dan Information Gain untuk Identifikasi Fake Account di Twitter


Aplikasi ini merupakan implementasi dari rancangan sistem secara bertahap dalam pembuktian perhitungan optimasi algoritma SVM dengan bantuan seleksi fitur IG dan ekstraksi fitur LDA guna mengidentifikasi fake account di sosial media Twitter. Data set dikumpulkan dengan metode crawling dengan bantuan modul Tweepy sebanyak 2000 data awal termasuk duplikasi pengguna selama 6 bulan terakhir dengan 21 fitur berbeda.

Tahapan Perhitungan
Tentang Dataset
1. Pengumpulan data dengan teknik crawling
2. Persiapan data dengan preprocessing
3. Seleksi fitur menggunakan IG
4. Ekstraksi fitur menggunakan LDA
5. Identifikasi label dan mengukur akurasi SVM

  • Jumlah duplikasi data = 633 data
  • Jumlah data akhir = 1367 data
